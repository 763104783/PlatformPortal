## 1 设计规范

### 1.1 主题风格

- **主题色**：紫荆花紫色（#7B2D8E）
- **风格**：科技 + 军事实战风格
- **背景**：深色科技风底色（#0D0B1A），配合光效粒子感

### 1.2 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #7B2D8E | 紫荆花紫，按钮/链接/高亮 |
| 主色浅 | #9B4DCA | 悬停/渐变 |
| 主色深 | #5A1D6E | 按下状态 |
| 背景深色 | #0D0B1A | 页面底色 |
| 背景卡片 | rgba(255,255,255,0.05) | 毛玻璃卡片 |
| 文字主色 | #FFFFFF | 标题/正文 |
| 文字次色 | rgba(255,255,255,0.7) | 辅助说明 |
| 成功色 | #67C23A | 成功状态 |
| 警告色 | #E6A23C | 警告状态 |
| 危险色 | #F56C6C | 删除/错误 |
| 信息色 | #909399 | 禁用/次要 |

### 1.3 字体规范

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| H1 | 36px | Bold | 平台名称 |
| H2 | 28px | Bold | 页面标题 |
| H3 | 22px | SemiBold | 区块标题 |
| 正文 | 14px | Regular | 正文内容 |
| 辅助 | 12px | Regular | 说明文字 |

### 1.4 间距与圆角

- 页面内边距：24px
- 卡片间距：16px
- 卡片圆角：12px
- 按钮圆角：6px
- 输入框圆角：6px

## 3 各页面详细设计

### 3.1 门户欢迎页

**布局**：左右分栏

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   ┌─────────────────────┐  ┌─────────────────────────┐ │
│   │                     │  │                         │ │
│   │    [平台Logo]       │  │   ┌─────────────────┐   │ │
│   │                     │  │   │  产品展示入口   │   │ │
│   │    XX平台           │  │   └─────────────────┘   │ │
│   │    赋能平台门户      │  │                         │ │

- 背景使用科技风深色+光效

│ ▼ 业务编排│  │  └──────┘ └──────┘ └──────┘          │  │

│   类型2  │  │  ┌──────┐ ┌──────┐ ┌──────┐          │  │
│          │  │  │ 卡片4 │ │ 卡片5 │ │ 卡片6 │          │  │
│ ▼ 模型管理│  │  └──────┘ └──────┘ └──────┘          │  │

│          │  └────────────────────────────────────────┘  │
└──────────┴──────────────────────────────────────────────┘
```

**侧边栏树形结构：**
```
应用交付
├── 应用类型1
├── 应用类型2
└── 应用类型3

├── 业务类型1
├── 业务类型2
└── 业务类型3
模型管理
├── 数据实体
└── 算法能力
```

**应用交付区：**
- 支持卡片/列表两种布局切换
- 卡片展示：封面图、名称、简介、版本号
- 点击卡片 → 新窗口打开应用链接
- 支持按名称搜索、按类型筛选

**业务编排区：**
- 支持卡片/列表两种布局切换
- 卡片展示：名称、简介、关联模板、制作人
- 点击卡片 → 查看流程详情（弹窗或跳转）

**模型管理区：**
- 列表布局展示
- 列：序号、名称、简介、制作人、版本、制作时间
- 点击行 → 查看模型详情

## 5 交互规范

### 5.1 操作反馈

| 操作 | 反馈方式 |
|------|----------|
| 新增/编辑成功 | ElMessage.success 提示 |
| 删除确认 | ElMessageBox.confirm 弹窗 |
| 批量删除确认 | ElMessageBox.confirm + 显示数量 |
| 表单校验失败 | 表单项红色提示 |
| 接口错误 | ElMessage.error 提示 |
| 数据加载中 | v-loading 遮罩 |

### 5.2 跳转规范

| 场景 | 方式 |
|------|------|
| 打开业务APP | window.open 新窗口 |
| 页面内导航 | Vue Router push |
| 查看详情 | 弹窗/抽屉 或 路由跳转 |
| 返回上一页 | router.back() |

### 5.3 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| 大屏 | ≥1920px | 主要适配目标 |
| 标准 | 1366-1919px | 笔记本屏幕 |
| 最小 | ≥1200px | 最低支持 |

## 6 科技军事风设计元素

### 6.1 视觉元素

- **背景**：深色底 + 微光粒子/网格线
- **卡片**：毛玻璃效果（backdrop-filter: blur）
- **边框**：1px 紫色发光边框
- **标题**：渐变色文字（紫→蓝）
- **按钮**：紫色渐变 + 悬停发光
- **图标**：线性科技风图标

### 6.2 动效

- 卡片悬停：轻微上浮 + 阴影加深
- 页面切换：淡入淡出
- 数据加载：骨架屏/进度条
- 入口按钮：呼吸光效
