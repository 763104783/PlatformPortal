## 1 概述

### 1.1 数据库选型

- **数据库**：SQLite 3.x
- **文件名**：`platform.db`
- **特点**：嵌入式、零配置、单文件存储，适合内网小规模系统

### 1.2 设计原则

- 所有表使用自增主键 `id`
- 时间字段统一使用 `TEXT` 类型存储 ISO 格式（`yyyy-MM-dd HH:mm:ss`）
- 逻辑删除使用 `status` 字段（1=启用，0=禁用）
- 枚举量数据独立建表，支持动态管理
- 多对多关系使用中间关联表

## 3 表结构设计

### 3.1 sys_user（系统用户表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 主键 |
| username | TEXT | NOT NULL, UNIQUE | 用户名 |
| password | TEXT | NOT NULL | 密码（加密存储） |
| role | TEXT | NOT NULL, DEFAULT 'USER' | 角色：ADMIN/USER |
| status | INTEGER | DEFAULT 1 | 状态：1启用/0禁用 |
| create_time | TEXT | | 创建时间 |
| update_time | TEXT | | 更新时间 |

```sql
CREATE TABLE IF NOT EXISTS sys_user (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT NOT NULL UNIQUE,
    password TEXT NOT NULL,
    role TEXT NOT NULL DEFAULT 'USER',
    status INTEGER DEFAULT 1,
    create_time TEXT DEFAULT (datetime('now','localtime')),
    update_time TEXT DEFAULT (datetime('now','localtime'))
);
```

### 3.8 process_template_rel（流程-模板关联表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 主键 |
| process_id | INTEGER | NOT NULL, FK | 业务流程ID |
| template_id | INTEGER | NOT NULL, FK | 流程模板ID |

```sql

    id INTEGER PRIMARY KEY AUTOINCREMENT,
    process_id INTEGER NOT NULL,
    template_id INTEGER NOT NULL,
    FOREIGN KEY (process_id) REFERENCES biz_process(id) ON DELETE CASCADE,
    FOREIGN KEY (template_id) REFERENCES process_template(id) ON DELETE CASCADE
);
```

| create_time | TEXT | | 创建时间 |

```sql
CREATE TABLE IF NOT EXISTS delivery_app (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    app_type_id INTEGER,
    url TEXT NOT NULL,
    description TEXT,
    cover_image TEXT,
    detail TEXT,
    version TEXT DEFAULT '1.0.0',
    click_count INTEGER DEFAULT 0,
    sort_order INTEGER DEFAULT 0,
    status INTEGER DEFAULT 1,
    create_time TEXT DEFAULT (datetime('now','localtime')),
    update_time TEXT DEFAULT (datetime('now','localtime')),
    FOREIGN KEY (app_type_id) REFERENCES app_type(id) ON DELETE SET NULL
```

## 4 索引设计

```sql
-- 用户表
CREATE INDEX IF NOT EXISTS idx_user_username ON sys_user(username);
CREATE INDEX IF NOT EXISTS idx_user_role ON sys_user(role);

-- 交付应用表
CREATE INDEX IF NOT EXISTS idx_app_type ON delivery_app(app_type_id);
CREATE INDEX IF NOT EXISTS idx_app_status ON delivery_app(status);
CREATE INDEX IF NOT EXISTS idx_app_name ON delivery_app(name);

-- 业务流程表
CREATE INDEX IF NOT EXISTS idx_process_name ON biz_process(name);

-- 模型表
CREATE INDEX IF NOT EXISTS idx_data_model_name ON data_model(name);
CREATE INDEX IF NOT EXISTS idx_algo_model_name ON algo_model(name);

-- 宣贯数据表
CREATE INDEX IF NOT EXISTS idx_showcase_category ON showcase_data(category);

-- 关联表
CREATE INDEX IF NOT EXISTS idx_app_process_app ON app_process_rel(app_id);
CREATE INDEX IF NOT EXISTS idx_app_process_process ON app_process_rel(process_id);
CREATE INDEX IF NOT EXISTS idx_process_tpl_process ON process_template_rel(process_id);
CREATE INDEX IF NOT EXISTS idx_process_tpl_template ON process_template_rel(template_id);
```
