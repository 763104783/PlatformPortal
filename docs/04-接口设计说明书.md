## 1 接口规范

### 1.1 基础约定

| 项目 | 说明 |
|------|------|
| 基础路径 | `/api` |
| 数据格式 | JSON（`Content-Type: application/json`） |
| 认证方式 | 请求头 `Authorization: {token}` |
| 字符编码 | UTF-8 |

### 1.2 统一响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 1.3 错误码定义

| code | 含义 | 说明 |
|------|------|------|
| 200 | 成功 | 请求处理成功 |
| 400 | 参数错误 | 请求参数不合法 |
| 401 | 未认证 | Token无效或过期 |
| 403 | 权限不足 | 非管理员访问管理接口 |
| 500 | 服务器错误 | 内部异常 |

### 1.4 分页参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| page | int | 1 | 页码（从1开始） |
| size | int | 10 | 每页条数 |

**分页响应：**
```json
{
  "code": 200,
  "data": {
    "records": [],
    "total": 100,
    "page": 1,
    "size": 10
  }
}
```

## 2 认证接口

### 2.1 用户登录

```
POST /api/auth/login
```

**请求体：**
```json
{
  "username": "admin",
  "password": "admin123"
}
```

**响应：**
```json
{
  "code": 200,
  "data": {
    "token": "uuid-token-string",
    "userId": 1,
    "username": "admin",
    "role": "ADMIN"
  }
}
```

### 2.2 获取当前用户信息

```
GET /api/auth/info
```

**请求头：** `Authorization: {token}`

**响应：**
```json
{
  "code": 200,
  "data": {
    "id": 1,
    "username": "admin",
    "role": "ADMIN"
  }
}
```

### 2.3 用户登出

```
POST /api/auth/logout
```

**请求头：** `Authorization: {token}`

## 3 平台配置接口

### 3.1 获取平台配置（公开）

```
GET /api/config/list
```

**响应：**
```json
{
  "code": 200,
  "data": [
    { "configKey": "platform_name", "configValue": "XX平台" },
    { "configKey": "platform_logo", "configValue": "/uploads/logo.png" },
    { "configKey": "company_name", "configValue": "XX公司" },
    { "configKey": "platform_version", "configValue": "1.0.0" },
    { "configKey": "platform_intro", "configValue": "基于模型驱动架构的标准赋能平台" }
  ]
}
```

### 3.2 更新平台配置（管理员）

```
PUT /api/config/update
```

**请求体：**
```json
{
  "platform_name": "XX赋能平台",
  "company_name": "XX科技公司",
  "platform_version": "2.0.0"
}
```

### 3.3 文件上传（管理员）

```
POST /api/config/upload
Content-Type: multipart/form-data
```

| 参数 | 类型 | 说明 |
|------|------|------|
| file | File | 图片文件（≤5MB） |
| fileKey | String | 配置键名（如 platform_logo） |

**响应：**
```json
{
  "code": 200,
  "data": "/uploads/xxxx-uuid.png"
}
```

## 9 交付应用接口

### 9.1 分页查询应用列表（公开）

```
GET /api/app/page?page=1&size=10&keyword=xxx&typeId=1&sortField=clickCount
```

| 参数 | 说明 |
|------|------|
| keyword | 按名称/简介模糊搜索 |
| typeId | 按应用类型筛选 |
| sortField | 排序字段：clickCount/name/sortOrder |

**响应记录含关联信息：**
```json
{
  "id": 1,
  "name": "智能报表系统",
  "url": "http://report.example.com",
  "description": "智能数据报表",
  "coverImage": "/uploads/app1.png",
  "version": "2.0.0",
  "clickCount": 128,
  "appType": { "id": 1, "name": "数据分析类" },
  "processes": [
    { "id": 1, "name": "数据处理流程" }

}
```
