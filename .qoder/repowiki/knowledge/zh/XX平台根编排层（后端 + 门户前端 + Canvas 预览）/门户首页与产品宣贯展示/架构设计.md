三个同级 Vue 单文件组件，分别对应路由 /、/showcase、/showcase/:id：
- PortalHome.vue：门户入口页，通过 getConfigs 拉取平台配置（configKey→configValue 扁平化），通过 getStatsOverview 加载应用数、访问量等统计，并依据 userStore.isAdmin 切换管理后台按钮。
- Showcase.vue：产品宣贯列表页，内置 dimensions 常量定义五大维度键值，使用 el-radio-group 切换 activeCategory 后调用 getShowcaseItems 渲染卡片网格；同时用 ECharts 在 mounted + nextTick 后初始化环形饼图，并行请求各维度数据计算占比。
- ShowcaseDetail.vue：宣贯详情页，通过 useRoute().params.id 读取路由参数，调用 getShowcaseDetail 获取 item，并用 computed 将 category 映射为中文标签显示。
依赖方向单一：仅向上依赖 ../api 与 ../stores/user，无跨视图通信，状态完全由响应式 ref 与组合式 API 驱动。
